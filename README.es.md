# 🤖 AI Agent Starter (Español)
### Boilerplate de agente LLM listo para producción con llamado de herramientas (tool-calling), memoria y streaming.

---

## 🚀 Inicio Rápido
Este repositorio proporciona una base sólida para construir agentes de IA sofisticados utilizando los últimos modelos y patrones arquitectónicos.

---

## 🛠️ Características
- **Tool-Calling:** Integración nativa para que los agentes ejecuten funciones.
- **Memoria:** Persistencia de contexto para conversaciones de largo plazo.
- **Streaming:** Respuestas en tiempo real para una mejor experiencia de usuario.
